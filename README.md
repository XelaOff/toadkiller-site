# Toad Killer

$TOAD - The alpha predator of memecoins.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Run the app:
   ```
   npm run dev
   ```
3. Build for production:
   ```
   npm run build
   ```
